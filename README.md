# Crossing Road

## Vài điều lưu ý

- Clone về hết luôn, t có để cái .sln để có thể bật lên luôn ấy

- Mới làm phần sườn, còn nhiều thứ chưa làm xong nữa

- Nếu được thì lưu source vào đây luôn cho dễ ghi nhớ
